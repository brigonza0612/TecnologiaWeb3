HyperText Markup Language


HTML, conocido como HyperText Markup Language, es el lenguaje base con el que se construye la estructura de cualquier página web. Su función principal es organizar el contenido: definir dónde se ubican los textos, imágenes, enlaces, títulos, tablas y otros elementos dentro de un sitio. Los navegadores interpretan este código y lo convierten en la página visual que el usuario finalmente observa.

Aunque HTML no es un lenguaje de programación tradicional ya que no utiliza variables, condiciones o ciclos es totalmente esencial, pues establece la forma y la semántica de cada componente de una web.

Para organizar el contenido, HTML utiliza marcas o etiquetas, que funcionan como instrucciones que el navegador debe seguir para mostrar correctamente cada parte del documento. Cada etiqueta tiene un propósito específico y contribuye a la estructura lógica del sitio. Por ejemplo:

  • <h1> a <h6> se utilizan para crear títulos y subtítulos, desde el más importante (h1) hasta niveles más específicos (h6)
  • <p> define un párrafo de texto.
  • <a> sirve para crear enlaces que permiten navegar entre páginas o dirigir al usuario a recursos externos.
  • <img> se usa para insertar imágenes dentro del contenido.
  • <table> permite construir tablas para organizar datos de manera ordenada.

Además de estas etiquetas básicas, HTML cuenta con muchas otras marcas que permiten crear secciones completas, insertar contenido multimedia o estructurar documentos más complejos. Algunas de ellas son:

<head> <title> <body> <header> <footer> <article> <section> <div> <span> <aside> <audio> <video> <canvas> <ul> <ol> <li> <nav> <details> <embed> <progress> entre muchas más.

Gracias al uso adecuado de estas etiquetas, las páginas web logran una estructura clara, coherente y semántica, lo que mejora la experiencia del usuario, facilita la accesibilidad y permite que los motores de búsqueda interpreten correctamente la información que presentan.
