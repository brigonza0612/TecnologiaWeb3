Lenguajes del desarrollo web


Tabla Comparativa entre Html, Css y JavaScript


Lenguaje Rol Función Tipo de lenguaje Uso
HTML Estructura Define el contenido y la organización de la página. Lenguaje de marcado Títulos, parrafos, imaganenes, tabla, formularios
CSS Presentacion Controla el diseño, los colores, la tipografías y disposicion visual de los elementos. Lenguaje de estilos Colores, fuentes, márgenes, diseños con flexbox o grid.
JavaScript Interactividad Permite crear páginas dinámicas que reaccionan a las acciones del usuario. Lenguaje de programación Menús desplegables, animaciones, validación de formularios, actualizaciones sin recargar la página.
